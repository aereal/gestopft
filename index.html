gestopft

Description

Framework for CLI Application (like App::CLI)

Installation

Archive

bundle install
bundle exec rake install

Gem

gem install gestopft

Synopsis

require "gestopft"

class Echo < Gestopft::App
  option :verbose, "print debug messages."

  def echo(message)
    puts "Echo#echo is invoked." if @options[:verbose]
    puts message
  end
end
